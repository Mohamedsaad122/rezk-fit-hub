# Active Sessions & Remote Revocations

Rezk Fit Hub logs every user login session in an encrypted, queryable state.

## Controls

### 1. Active Logins query
Users can view details for all connected devices:
- **IP Address**
- **User Agent & Device**
- **Created date & Expiration time**

### 2. Revocation & remote logout
Allows immediate termination of remote connections by ID.

### 3. Revoke All Other Devices
Forces logout on all remote sessions while keeping the current device active.
