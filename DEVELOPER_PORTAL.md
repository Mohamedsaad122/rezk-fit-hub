# Developer Portal & Platform Documentation

The Developer Portal provides a secure interface for gym partners and third-party integrations to connect with Rezk Fit Hub.

## Views

- **Developer Dashboard**: Overview of integrations, total calls, and apps.
- **API Keys Manager**: Allows generating and rotating API keys with precise security scopes.
- **OAuth Applications**: Regulates client secrets and redirection URLs.
- **GraphQL Playground**: Includes an interactive editor, schema doc explorer, and mock returns.
- **API Logs Panel**: Displays latency metrics and gateway responses.
- **Usage Statistics Dashboard**: Renders interactive charts showing usage spikes.
- **SDK Downloader**: Allows downloading customized, ready-to-run wrappers.
