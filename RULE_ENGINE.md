# Rule & Formula Evaluation Engine

Calculates IF/ELSE switch rules, logical boolean expressions, priority levels, and dynamic math expressions.

## Expression Operators
- `EQUALS`
- `NOT_EQUALS`
- `GREATER_THAN`
- `LESS_THAN`
- `CONTAINS`

## Dynamic Math Formulas
Parses math expressions safely:
- Sandbox evaluation isolates context parameters (e.g., `x * 1.15`).
- Multi-variable substitution.
- Formula sandbox testing interface in [RuleEngine.jsx](file:///d:/projects/rezk%20project/rezk-fit-hub/src/pages/RuleEngine.jsx).
